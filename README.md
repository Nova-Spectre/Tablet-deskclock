\# Tablet Dashboard for Tab



Beautiful dashboard with clock, weather, stopwatch, timer, and reminders.



\## Features

\- 12-hour clock format

\- Live weather (Mumbai)

\- 5 beautiful themes with animations

\- Stopwatch, Timer, Reminders

\- Light \& Dark modes



\## Deploy

1\. Run: npm install

2\. Run: npm run build

3\. Deploy dist folder to Netlify/Vercel
